Formularios

Ejemplos y pautas de uso para estilos de control de formulario, opciones de diseño y componentes personalizados para crear una amplia variedad de formularios.

Tipos

Un input estandar, define un campo de texto de una línea.

Define un input de contraseña que esconde los caracteres ingresados.

Se utiliza para inputs que deben contener fecha y hora, permite ingresarla o seleccionarla.

Se utiliza para inputs que deben contener una fecha, permite ingresarla o seleccionarla.

Un input donde ingresar un horario.

Un input que permite al usuario ingresar o seleccionar un mes y un año.

Un input donde ingresar o seleccionar una semana y un año.

Define un checkbox.

Los checkbox permiten a un usuario seleccionar CERO o MÁS opciones de un número limitado de opciones.

Los radio permiten a un usuario seleccionar SÓLO UNA opcion de un número limitado de opciones.
