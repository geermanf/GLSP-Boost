Tablas

Las tablas son elementos muy utilizados en sitios y/o sistemas como los que se desarrollan en la empresa. Se utilizan para mostrar ordenadamente listados de datos, que mucha veces son demasiado largos y complejos. Además de poder tener una visiòn mucho más ordenada y simple de tantos datos, las tablas pueden a su vez filtrar los datos según las necesidades del usuario, haciendo este listado más concreto y específico. También pueden ordenarse los datos según le convenga al usuario y editar los datos allí mismo.
