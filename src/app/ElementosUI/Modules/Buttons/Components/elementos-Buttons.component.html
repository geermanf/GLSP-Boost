Botones

Un botón indica una posible acción del usuario

Tipos

Un botón estandar

Un botón puede ser configurado para mostrar diferentes niveles de énfasis

Un botón puede animarse para mostrar contenido oculto

Un botón puede aparecer junto a una etiqueta

Un botón sólo puede tener un ícono

Un botón puede utilizar un ícono como una etiqueta

Un botón básico es menos pronunciado

Se puede configurar un botón para que aparezca en fondos oscuros

Grupos

Los botones pueden existir juntos como un grupo

Los grupos de botones pueden mostrar grupos de íconos

Contenido

Los grupos de botones pueden contener condicionales

Estados

Un botón puede mostrar la selección actual del usuario

Un botón puede mostrar que actualmente no se puede interactuar

Un botón puede mostrar un indicador de carga

Variaciones

Un botón se puede configurar para estar vinculado a una red social

Un botón puede tener diferentes tamaño

Un botón se puede alinear a la izquierda o a la derecha de su contenedor

Un botón puede tener diferentes colores

Un botón puede reducir su relleno para encajar en un espacio más apretado

Un botón se puede formatear para activar y desactivar

Un botón puede indicar una consecuencia positiva

Un botón puede indicar una consecuencia negativa

Un botón puede tomar el ancho de su contenedor

Un botón puede ser circular

Un botón puede ser conectado a la parte superior o inferior de otro contenidor

Se puede adjuntar un botón a la izquierda oa la derecha de otro contenido

Variaciones en grupo

Los grupos pueden ser configurados para que aparezcan verticalmente

Los grupos se pueden formatear como icono

Los grupos se pueden configurar como iconos etiquetados

Los grupos se pueden configurar para utilizar varios tipos de botones juntos

Los grupos pueden tener sus anchos divididos uniformemente

Los grupos pueden compartir un color

Un grupo de botones puede ser menos pronunciado

Los grupos pueden tener un tamaño compartido
